## 开始

这是一份在本地构建项目的指导的例子。
要获取本地副本并且配置运行，你可以按照下面的示例步骤操作。

### 依赖

* Node.js v20.12

### 安装


1. 克隆本仓库
   ```sh
   git clone https://github.com/Lim-Watt/iec.git
   ```
3. 安装 NPM 包
   ```sh
   npm install
   ```
4. 在 `config.json` 中填写你的 OS 和用户名
   - OS 必须是 `linux` 或 `win`。
   - 用户名是聊天时显示的名称。
